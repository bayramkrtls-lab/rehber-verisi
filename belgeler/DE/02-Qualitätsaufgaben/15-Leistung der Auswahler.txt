Die SAs werden sich an Sie wenden, um sich nach ihrer Auswahlsleistung des Tages zu erkundigen. Sie können eine Liste in Excel erstellen. SCC -> Übersicht -> Auswahl  -> Alle Filter löschen -> Klicken Sie auf die Option "Exportieren nach CSV" in der rechten oberen Ecke des Bildschirms -> Öffnen Sie die heruntergeladene Excel-Datei -> Wählen Sie Spalte A aus -> Klicken Sie auf "Daten" -> "Text in Spalten" -> Wählen Sie "Getrennt" aus -> Klicken Sie auf "Weiter" -> Wählen Sie "Komma als Trennzeichen" aus -> Klicken Sie auf "Weiter" -> "Fertig stellen" -> Wählen Sie die gesamte Tabelle aus -> Klicken Sie auf "Einfügen" -> Klicken Sie auf "PivotTable" -> Klicken Sie auf "OK" -> Wählen Sie auf der rechten Seite "Auswahlslistencode und Zuordnung" aus Ziehen Sie den Auswahlslistencode auf die Registerkarte "Werte" -> Klicken Sie auf den Pfeil in den Zeilenbeschriftungen -> Wählen Sie "Beschriftungsfilter" aus -> Wählen Sie die Option "Enthält nicht" aus Geben Sie "," ein -> Klicken Sie auf "OK" aus Klicken Sie mit der rechten Maustaste auf eine beliebige Anzahl von Auswahlslistencodes neben den Namen -> Wählen Sie "Sortieren" -> "Sortieren absteigend".
