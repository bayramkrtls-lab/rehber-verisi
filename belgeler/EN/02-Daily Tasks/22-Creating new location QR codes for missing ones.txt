Connect your Zebra 620 printer to your computer using the USB cable that printer has -> Open the Excel spreadsheet named SzPrinter -> Enter the missing QR codes into the spreadsheet -> Click on Generate Labels -> Select 300 dpi -> In your computer's search bar, type "printers and scanners" -> Click on ZDesigner ZD620-300dpi ZPL -> Printers properties -> Driver settings -> Settings -> Click on the three dots next to Send file and select the first option, the generated QR code file -> Click Send
