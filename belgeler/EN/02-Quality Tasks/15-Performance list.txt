SAs will come to you to ask about their pick performance for that day. You can make a list in Excel. SCC -> Overview -> Pick -> Clear all filters -> Click on the Export to CSV option on the right corner of the screen -> Open the downloaded Excel file -> Select Column A -> Click on Data(Datei) -> Text to Columns(Text im Spalten) -> Choose Deliminated -> Click Next -> Select Comma on Delimiters -> Next -> Finish(Fertig) -> Select all table -> Click on Insert -> Click on Pivot Table -> Click on OK -> Select Picklist Code and Associate on the right side -> Drag the picklist code to the values tab -> Click on the arrow in Row labels -> Choose Label Filters(Beschriftungsfilter) -> Choose Does not contain(Enhält nicht) option -> Type “,” -> Click OK -> Click right on any of the count of picklist code number by the names -> Select Sort->Sort Largest to Smallest.
